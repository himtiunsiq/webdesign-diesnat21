Tentang Design Website

Design ini dibuat menggunakan NATIVEHTML, CSS, dan JS Tanpa Library apapun, mengukuti aturan dari panitia (TIDAK BOLEH menggunakan library pihak ketiga seperti Bootstrap, Bulma, Tailwind, jQuery, dll).


Design ini telah responsive pada perangkat mobile dan dasktop, Namun saya masih menemukan beberapa Bug seputar Responsivenya, dan saya masih belum memperbaiki beberapa bug tersebut karena keterbatasan waktu. Berikut rekomendasi untuk ukuran pengujian Resposive design.


Ukuran yang disarankan saat pengujian :

Mobile : (360 x 780),(360 X 800),(375 x 812) dan {393 x 851).

Desktop : (1280 x 800) .
